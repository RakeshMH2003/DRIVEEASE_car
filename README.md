# 🚗 DriveEase — Premium Car Rental Platform

## 📌 Project Overview

**DriveEase** is a full-stack-style web application designed for **car rental management**.
It connects **Users, Vendors, and Admins** in a single platform to enable seamless vehicle booking, management, and approval.

The system provides:

* Easy vehicle browsing & booking
* Vendor vehicle listing & management
* Admin approval & monitoring system

---

## ✨ Features

### 👤 User Module

* User Registration & Login
* Browse vehicles with filters (SUV, Sedan, Luxury, etc.)
* Book vehicles instantly
* View booking history
* Profile management
* Contact admin support

---

### 🏢 Vendor Module

* Vendor Registration & Login
* Add vehicles with images
* Manage vehicle listings
* View bookings
* Pending approval system (admin controlled)

---

### 🛠️ Admin Module

* Admin dashboard
* Approve/Reject vendors
* Manage vehicles
* Monitor bookings
* Handle user messages

---

### 🚘 Vehicle Features

* Search & filter vehicles
* Sort by price, rating, name
* Trending vehicles display
* Detailed vehicle cards
* Image preview system

---

## 🧠 Technologies Used

### Frontend

* HTML5
* CSS3 (Modern UI with animations)

## ⚙️ How It Works

### 1. Data Management

* Uses **LocalStorage** to store:

  * Users
  * Vendors
  * Vehicles
  * Bookings
  * Messages

### 2. Authentication

* Separate login for:

  * Users
  * Vendors
  * Admin

### 3. Booking Flow

1. User browses vehicles
2. Selects vehicle
3. Chooses date
4. Confirms booking
5. Booking stored in LocalStorage
